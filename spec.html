Specification / Feature List

Graphical Analysis

  • Full 3D environment view, free orbit/zoom camera, WebGL.
  • Mast ribbon shows attitude and boat speed (highlights poor speed).
  • Wind ribbon shows wind strength (highlights poor strength), and current tack bias.
  • Velocity arrows show:
    • Speed through water and true wind angle
    • VMG and current tack bias angle
    • Shadow arrow shows GPS reported speed and direction
  • True wind line shows current true wind direction and strength.
  • Real time manoeuvre losses and gains calculated and animated.
  • Rudder data overlaid if available.
  • Video upload and recompression, sync taken from in-camera timestamps.
  • Zoom/Pan overview graph shows beats, runs, reaches and manoeuvres in one place.
  • Upwind/Downwind values shown:
    • True Windspeed
    • Apparent Wind Angle
    • Velocity Made Good
  • Average wind angle shown on per-tack basis.
  • VMG graphed against true wind angle both upwind and down.

Connectivity

  • NMEA 2000 → Bluetooth 4 → Mobile App → Cloud
  • NMEA 0183 → WiFi → Mobile App → Cloud
    • Bluetooth gateway supports circa 130 messages/sec
    • Mobile App for both iPhone and Android
    • Upload to cloud over WiFi or mobile data
  • Log import from B&G, Expedition, NMEA 2000 (CanBoat) and timestamped NMEA 0183
  • Post-import source filtering for NMEA 2000.
